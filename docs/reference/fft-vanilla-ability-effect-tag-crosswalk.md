## Purpose

This document is the effect-shaped view of the vanilla ability atlas.
Use it when a proposal touches a mechanic first, such as `status_add`, `ct_action`,
`equipment_break`, `global`, `reaction`, `undead`, or `movement`, and we need to see
every vanilla record that already belongs to that effect family.

Rows can appear under multiple tags. That is intentional: many FFT abilities combine
damage, status, timing, resource, terrain, or equipment behavior.

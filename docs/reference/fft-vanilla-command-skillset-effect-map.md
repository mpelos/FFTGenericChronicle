## Purpose

This document is the job/command-shaped view of the vanilla ability atlas.
Use it when redesigning one job at a time and checking what the original game already
expressed in that command family.

The rows inherit the same trust model as `fft-vanilla-ability-effect-index.md`: local
ID/name/JP/override flags are extracted facts; command ownership and effect tags are
research/design classifications for planning.

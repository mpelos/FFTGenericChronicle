## Integer Sanity Checks

These cells are recomputed by hand from the LT7 `DclDerivedVariables` chain. All divisions below are integer divisions with truncation.

- Dagger into Plate Mail: type `thrust`, armor `plate`. ST = 8+4 = 12; gurpsThr[12] = 8; gross = 8+3 = 11; DR = 8; penFloor = (11*200)//1000 = 2; penetrating = max(2, 11-8) = 3; wounded = (3*2)//1 = 6; trait = 760+(70*590)//100 = 1173; final = max(1, (6*1173)//1000) = 7.
- Rod into Clothing: type `crush`, armor `clothes`. ST = 8+4 = 12; gurpsSw[12] = 12; gross = 12+3 = 15; DR = 2; penFloor = (15*200)//1000 = 3; penetrating = max(3, 15-2) = 13; wounded = (13*1)//1 = 13; trait = 760+(70*590)//100 = 1173; final = max(1, (13*1173)//1000) = 15.
- Save the Queen into Plate Mail: type `cut`, armor `plate`. ST = 8+4 = 12; gurpsSw[12] = 12; gross = 12+18 = 30; DR = 9; penFloor = (30*200)//1000 = 6; penetrating = max(6, 30-9) = 21; wounded = (21*3)//2 = 31; trait = 760+(70*590)//100 = 1173; final = max(1, (31*1173)//1000) = 36.

Open questions:

- Should missile continue to use `gurpsSw` because LT7 says `if(thrust, thr, sw)`, or should ranged weapons eventually get their own base source?
- Should armor sharpness target the design note's medium matchup goal directly, or should first-pass tuning prioritize matching the LT7 live anchors?
- Should Brave remain a large offensive roster lever, or should weapon type and armor matchup carry more of the damage spread?

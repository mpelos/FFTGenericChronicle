## Native contract

The pass-2 Nature's Wrath branch computes `tileIndex = level*256 + y*mapWidth + x` from
the **reactor's own tile**, reads `tileTable[tileIndex].terrainType & 0x3F`, maps that
terrain id through the 64-byte table at RVA `0x67F750`, and stages the resulting normal
Geomancy ability id `126..137` as a typed order of kind `0x11`.

This is not a rider-free special damage action. It reuses the ordinary Geomancy payload,
whose ability data carries its normal status rider. The final DCL Nature's Wrath therefore
already has native own-tile terrain selection, but its `no status rider, no Scar` rule needs
reaction-aware status suppression. Global data removal would also remove the authored rider
from active Geomancy and is not equivalent.

## Implementation boundary

Damage can keep the selected payload id and flow through the existing per-ability formula
map. Status suppression can reuse the durable DCL status-output controls only after action
provenance distinguishes a Nature's Wrath reaction execution from an ordinary cast of the
same payload. Cadence remains a separate once-per-attacker-action commit concern.

## Native contract

The calc path copies the incoming order record, reads its exact ability id from `+2`, and
publishes it at word RVA `0x7B0778`. When Magick Counter `435` is selected, the reaction
dispatcher copies that word into reactor `unit+0x1E6`. Pass 2 then builds a typed order of
kind `0x0B` with `unit+0x1E6` as its payload. Native Magick Counter is therefore an exact
incoming-spell copy path, not a generic magic-flavoured basic attack.
